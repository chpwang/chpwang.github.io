Python 基本规则

  1. 区分大小写，即 case sensitive
  2. 空格重要，附带语意，即 spacing matters
  3. 运算符号有优先级（比如乘除法 * / 优先级高于加减法 +-）
  4. 运算的优先级可以使用括号来确定：优先级方向为从里往外
  5. Python 赋值时是「传值」效果，而不是「传址」效果

Python 中的注释

  • 以 # 开头的行
  • 以 """（三个双引号）包裹的代码块（Code Block）
