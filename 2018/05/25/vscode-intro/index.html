上手 Visual Studio Code

  2018-05-25     Tools

简介

VS Code 是微软开发的一款编辑器，迭代速度很快 ，产品定位是介于 IDE 和纯编辑器（比如 vi）之间，更多八卦可收听Teahour 91期 的介绍

VS Code 安装

安装很简单，从官网下载 App 正常安装即可，实在不懂就看官方教程

一般性设置

  • 设置从终端（Terminal）里打开 VS Code ：

    使用⇧⌘P打开 Command Palette，输入 shell command 后搜索安装 code 这个命令即可

    安装好后，使用以下方式从终端打开

    # 单纯打开 VS Code
$ code

# 使用 VS Code 打开指定路径的文件
$ code [文件路径]

# Example - 使用 VS Code 打开当前文件夹
$ code .

  • 设置使用 Emacs 快捷键（光标移动，Kill 和 Yank 等操作）：

    同样使用⇧⌘P打开 Command Palette，搜索安装 Emacs Friendly Keymap

个性化设置

VS Code 的设置通过添加代码指令来实现，有以下三个不同优先级的设置文件，它们之间是继承关系：

  1. Default User Settings - 基础 Settings
  2. User Settings - 覆盖上述 Default Settings
  3. Workspace Settings - 覆盖以上两者

设置「Editor 和 Terminal 间切换」的快捷键

// Toggle between terminal and editor focus
{ "key": "ctrl+`", "command": "workbench.action.terminal.focus"},
{ "key": "ctrl+`", "command": "workbench.action.focusActiveEditorGroup", "when": "terminalFocus"}

简单操作

  • 插件

    安装 Code Runner 插件来运行主流语言的代码：「menu」-> 「view」-> 「extension」或者使用快捷键 Cmd + Shift + X 来进入插件搜索界面。搜索关键词「Code Runner」然后点击安装即可。

  • 运行代码

    点击界面右上角的三角形 Run Code 按钮或者使用快捷键 Ctrl + Option + N 来运行的代码。

常用快捷键

注释 / 取消注释某一行代码（光标位于这一行后再按快捷键组合）：Ctrl + F + ;

有关 VS Code 的更多操作，可参考官方文档
