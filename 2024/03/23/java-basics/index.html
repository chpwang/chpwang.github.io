上手 Java

  2024-03-23     Programming Language

Java 是一门静态类型语言。对程序中各数据的类型有严格的检查。类型系统可以通过类型不匹配提前发现问题，保证代码可靠性。也可以在编译后，不用每次都询问一下数据的类型来程序提高性能。

Java 编程环境

  • 安装 JDK（可选）
    JDK（Java Development Kit）是用于开发 Java 应用程序的软件包，提供了编译（javac）、运行（JRE） Java 代码所需的工具和库。可以通过 Hombrew 来安装：

    ## 东西比较多，注意保持网络通畅
$ brew install openjdk

## brew install java
## 安装的是由 Apple 或者 Apple 授权的 Java 版本，通常是 Oracle JDK

## brew install openjdk 
## 安装的是由 OpenJDK 项目提供的开源 Java 版本，与 Oracle JDK 功能上是兼容的
## 不同的是 OpenJDK 是由社区维护的

    Tips：新手推荐在下文 IntelliJ IDEA 的 onboarding tour 教学过程中安装 JDK，省去各种复杂的配置

  • 安装 IntelliJ IDEA
    IntelliJ IDEA 是 JetBrains 公司推出的集成开发环境（IDE）。对 Java 编程来说，使用上比 VS Code 更加方便一些。对于初学者，安装 IntelliJ IDEA Community Edition 这个免费的版本就够用了。

    可以看到，IntelliJ IDEA 有极其友好的 onboarding 入门教程。点击 Start Tour in Java 试试吧。
    这个 Quick onbording tour 的过程还可以根据提示，非常方便地安装 JDK
